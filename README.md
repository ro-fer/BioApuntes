# 🧬 BioApuntes UNSAM

Banco colaborativo de apuntes, guías, parciales y recursos para estudiantes de Ingeniería Biomédica en UNSAM.

## ¿Qué es?

BioApuntes UNSAM es un índice comunitario para encontrar material de las materias de la carrera.

La idea es simple: buscar una materia y llegar rápido a apuntes, guías, parciales, finales, resueltos o links útiles.

## Sitio

https://ro-fer.github.io/BioApuntes/
## Importante

Este sitio no es oficial de la UNSAM.  
La información puede estar incompleta o desactualizada.  
Siempre verificá programas, fechas, correlativas y condiciones con la cátedra o canales oficiales.

## ¿Qué incluye?

- Banco de materias
- Links a Frubox, Drive y OneDrive
- Recursos importantes de la carrera
- Contactos útiles
- Formulario para colaborar

## Cómo colaborar

Podés aportar:

- Apuntes
- Guías
- Parciales
- Finales
- Resueltos
- Bibliografía
- Links útiles

Para colaborar, usá el formulario de aportes disponible en la web.
